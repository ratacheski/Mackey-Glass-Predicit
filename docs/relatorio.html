📚Interpretação Avançada das Métricas

Esta seção oferece uma análise mais profunda do significado de cada métrica no contexto do seu modelo de machine learning.

🎯 Como Interpretar os Resultados

  • R² próximo de 1: Indica que o modelo explica quase toda a variabilidade dos dados
  • RMSE baixo: Erros de predição pequenos em relação à escala dos dados
  • MAE baixo: Erro médio absoluto pequeno, mais robusto a outliers
  • MAPE baixo: Erro percentual pequeno, útil para comparação entre diferentes escalas

📊 Análises Disponíveis

📊 Q-Q Plot
Compara a distribuição dos resíduos com a distribuição normal. Pontos próximos à linha diagonal indicam normalidade dos resíduos.
📈 Análise de Distribuições
Compara as funções de distribuição (CDF) e densidade (PDF) entre valores reais e predições.
🔬 Teste Kolmogorov-Smirnov
Testa se as distribuições de valores reais e predições são estatisticamente similares.
🔄 Análise de Autocorrelação
Examina a dependência temporal nos dados, importante para séries temporais.

📚 Guia Completo de Métricas

🎓Aprenda sobre Métricas de Machine Learning

Este guia explica em detalhes todas as métricas utilizadas na avaliação dos modelos.

📊 R² (Coeficiente de Determinação)
O R² mede o quão bem o modelo explica a variabilidade dos dados.

Interpretação:
  • R² = 1.0: Predição perfeita
  • R² > 0.9: Excelente
  • R² > 0.8: Bom
  • R² > 0.6: Moderado
  • R² < 0.6: Baixo

Fórmula: R² = 1 - (SSres / SStot)
📏 RMSE (Raiz do Erro Quadrático Médio)
O RMSE mede a magnitude típica dos erros de predição.

Características:
  • Unidade igual aos dados originais
  • Penaliza erros grandes mais severamente
  • Valores menores indicam melhor performance
  • Sensível a outliers

Fórmula: RMSE = √(Σ(yi - ŷi)² / n)
📐 MAE (Erro Absoluto Médio)
O MAE mede o erro médio absoluto entre predições e valores reais.

Características:
  • Mais robusto a outliers que RMSE
  • Interpretação intuitiva
  • Unidade igual aos dados originais
  • Sempre não-negativo

Fórmula: MAE = Σ|yi - ŷi| / n
📈 MSE (Erro Quadrático Médio)
O MSE mede a média dos quadrados dos erros de predição.

Características:
  • Base para cálculo do RMSE
  • Penaliza erros grandes quadraticamente
  • Sempre não-negativo
  • Unidade quadrática dos dados originais

Fórmula: MSE = Σ(yi - ŷi)² / n
📊 MAPE (Erro Percentual Absoluto Médio)
O MAPE expressa o erro como percentual dos valores reais.

Interpretação:
  • MAPE < 5%: Excelente precisão
  • MAPE < 10%: Boa precisão
  • MAPE < 20%: Precisão moderada
  • MAPE ≥ 20%: Baixa precisão

Fórmula: MAPE = (100/n) × Σ|yi - ŷi|/|yi|
📊 EQMN1 (Erro Quadrático Médio Normalizado - Variância)
O EQMN1 normaliza o MSE pela variância dos valores reais.

Características:
  • Normalizado entre 0 e 1
  • Independente da escala dos dados
  • Útil para comparação entre datasets
  • Valores menores indicam melhor performance

Fórmula: EQMN1 = MSE / Var(y_true)
📉 EQMN2 (Erro Quadrático Médio Normalizado - Modelo Naive)
O EQMN2 normaliza o MSE pelo MSE de um modelo naive (persistence).

Características:
  • Compara com modelo naive que usa valor anterior
  • Ideal para séries temporais
  • Valores < 1 indicam que o modelo supera a persistência
  • Valores > 1 indicam que o modelo é pior que usar valor anterior

Fórmula: EQMN2 = MSE / MSE_naive
Interpretação:
  • < 0.5: Muito superior ao modelo naive
  • 0.5-1.0: Superior ao modelo naive
  • = 1.0: Equivalente ao modelo naive
  • > 1.0: Inferior ao modelo naive

💡Dicas de Interpretação

  • Use múltiplas métricas: Cada métrica oferece uma perspectiva diferente
  • Considere o contexto: A importância de cada métrica depende da aplicação
  • Analise os resíduos: Padrões nos erros podem revelar problemas no modelo
  • Compare com baselines: Avalie se o modelo supera métodos simples
  • Valide em dados novos: Performance em teste é crucial

📚 Metrics Guide

🎓Learn About Machine Learning Metrics

This guide explains all metrics used in model evaluation in detail.

📊 R² (Coefficient of Determination)
R² measures how well the model explains the data variability.

Interpretation:
  • R² = 1.0: Perfect prediction
  • R² > 0.9: Excellent
  • R² > 0.8: Good
  • R² > 0.6: Moderate
  • R² < 0.6: Low

Formula: R² = 1 - (SSres / SStot)
📏 RMSE (Root Mean Square Error)
RMSE measures the typical magnitude of prediction errors.

Characteristics:
  • Same unit as original data
  • Penalizes large errors more severely
  • Lower values indicate better performance
  • Sensitive to outliers

Formula: RMSE = √(Σ(yi - ŷi)² / n)
📐 MAE (Mean Absolute Error)
MAE measures the mean absolute error between predictions and actual values.

Characteristics:
  • More robust to outliers than RMSE
  • Intuitive interpretation
  • Same unit as original data
  • Always non-negative

Formula: MAE = Σ|yi - ŷi| / n
📈 MSE (Mean Square Error)
MSE measures the mean of squared prediction errors.

Characteristics:
  • Base for RMSE calculation
  • Penalizes large errors quadratically
  • Always non-negative
  • Squared unit of original data

Formula: MSE = Σ(yi - ŷi)² / n
📊 MAPE (Mean Absolute Percentage Error)

Formula: MAPE = (100/n) × Σ|yi - ŷi|/|yi|

Characteristics:

  • Lower values indicate better performance

Formula: NMSE1 = MSE / Var(y_true)

💡Interpretation Tips

  • Use multiple metrics: Each metric offers a different perspective
  • Consider the context: The importance of each metric depends on the application
  • Analyze residuals: Patterns in errors can reveal problems in the model
  • Compare with baselines: Evaluate if the model outperforms simple methods
  • Validate on new data: Performance on test is crucial

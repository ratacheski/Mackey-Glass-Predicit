üìä NMSE1 (Normalized Mean Square Error - Variance)
NMSE1 normalizes MSE by the variance of actual values.

Characteristics:
  • Normalized between 0 and 1
  • Scale-independent
  • Useful for comparison across datasets
  • Lower values indicate better performance

Formula: NMSE1 = MSE / Var(y_true)
üìâ NMSE2 (Normalized Mean Square Error - Naive Model)
NMSE2 normalizes MSE by the MSE of a naive model (persistence).

Characteristics:
  • Compares with naive model using previous value
  • Ideal for time series
  • Values < 1 indicate model outperforms persistence
  • Values > 1 indicate model performs worse than using previous value

Formula: NMSE2 = MSE / MSE_naive
Interpretation:
  • < 0.5: Much superior to naive model
  • 0.5-1.0: Superior to naive model
  • = 1.0: Equivalent to naive model
  • > 1.0: Inferior to naive model

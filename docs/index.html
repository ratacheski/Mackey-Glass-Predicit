Predição de Séries Temporais Mackey-Glass

📊 Sobre o Projeto

Este projeto implementa e compara três tipos de redes neurais (MLP, LSTM, GRU) para predição da série temporal Mackey-Glass, com múltiplas configurações e variações para análise abrangente.
